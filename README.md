## Automatic lawn mower

This project is a Gradle project to play with several automatic lawn mowers.

### Prerequisite
* Java 8 or higher
* Gradle

### Run with Gradle
```script
gradle run -Pfile=src/test/data-test.txt
```
### Gradle output
The last position of each lawn mowers

```script
> Task :run
1 3 N
5 1 E
```
